# S-007: Implement state divergence detection

**Epic:** E-002
**Priority:** P1

## Description

Implement detection of divergence between state.yaml and initiative config. When the dual-write contract is violated (e.g., manual edit, partial write failure), the system must detect the mismatch and report specific divergent fields.

## Acceptance Criteria

- [ ] Can compare state.yaml phase_status with initiative config phase_status
- [ ] Can compare state.yaml current_phase with initiative config current_phase
- [ ] Reports specific fields that diverge (not just 'files differ')
- [ ] Detection runs during pre-flight of every phase command
- [ ] Suggests `/sync` when divergence is detected
