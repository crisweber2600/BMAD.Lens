# S-001: Audit and remove auto-advance patterns

**Epic:** E-001
**Priority:** P0

## Description

Audit all existing phase workflow files in `_bmad/_config/custom/lens-work/workflows/` and identify any patterns that automatically advance to the next phase without requiring a PR merge gate. Remove all auto-advance patterns found. This is the highest-priority safety fix — it directly addresses Gap G and enforces FT1.

- [ ] Audit results documented with file paths and line numbers of removed patterns
