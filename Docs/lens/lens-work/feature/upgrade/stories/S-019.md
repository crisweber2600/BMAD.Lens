# S-019: Implement /switch command

**Epic:** E-005
**Priority:** P2
**Story Points:** 3
**Dependencies:** S-018, S-003

## Description

Implement the `/switch` command that changes the active initiative. Updates `state.yaml.active_initiative` pointer, reloads initiative context, and confirms the switch to the user.

## Acceptance Criteria

- [ ] Lists available initiatives if called without argument
- [ ] Switches active_initiative pointer in state.yaml
- [ ] Reloads initiative config for new active initiative
- [ ] Shows status of newly active initiative after switch
- [ ] Validates target initiative exists before switching
- [ ] Logs switch event in event log
