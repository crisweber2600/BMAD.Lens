# S-035: Implement /sync (remote-wins reconciliation)

**Epic:** E-009
**Priority:** P2

## Description

Implement the `/sync` command that reconciles local state with remote git state using remote-wins strategy. Fetches all remote branches, checks PR merge status, compares with local state, shows diff, applies with confirmation.

## Acceptance Criteria

- [ ] Fetches all remote branches
- [ ] For each initiative: checks phase branch existence and PR merge status
- [ ] Compares phase_status with git reality (merge-base --is-ancestor)
- [ ] Shows diff of proposed state changes before applying
- [ ] Applies changes only with user confirmation
- [ ] Logs state_sync event with changes_applied
