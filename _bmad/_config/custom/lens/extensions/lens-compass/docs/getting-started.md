# Getting Started with Lens Compass

Welcome to Lens Compass! This guide will help you get up and running.

---

## What This Module Does

Lens Compass provides role-aware guidance for Lens workflows. It learns who you are, tracks what you do, and suggests the right next step.

---

## Installation

```bash
bmad install lens
```

Lens Compass installs as a Lens extension by default.

---

## First Steps

1. Run onboarding to create your personal profile
2. Confirm your role(s) and preferences
3. Use Next Steps to receive workflow guidance

---

## Common Use Cases

- PO deciding between new-service and lens-switch
- Architect coordinating sprint planning
- Developer running dev-story and code-review
- Teams discovering who worked on a microservice
