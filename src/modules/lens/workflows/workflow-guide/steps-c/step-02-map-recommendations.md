# Step 2: Map Recommendations

## Goal
Map the lens and phase to recommended workflows.

## Instructions
- Use a simple mapping:
  - Domain: discovery, PRD, architecture
  - Service: architecture, epics/stories, implementation readiness
  - Microservice: implementation, testing, code review
  - Feature: dev story, QA, release checks
- Prefer workflows aligned to the detected BMAD phase.

## Output
- `recommendations`
